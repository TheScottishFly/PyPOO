## La programmation POO avec Python


Projet du cours du même nom sur OpenClassRooms.

Le but du projet est de réaliser un monde virtuel, comprenant des zones dans
lesquelles vivent des habitants ayant une position précise et des données uniques.
Les données des habitants proviennent de http://pplapi.com/.

L'objectif final est de pouvoir créer des graphiques de statistiques en se basant sur ces données.

L'objectif personnel est de valider la maîtrise de la POO en python.
